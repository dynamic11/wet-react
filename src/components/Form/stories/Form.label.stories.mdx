# Forms

## Form Label

`<Form.Label>` is used to label a form input.

### Visible Label

<Canvas>
  <Story name="Visible Label">
    <Form>
      <Form.Label htmlFor="inputPassword1">Password</Form.Label>
      <Form.Control type="password" id="inputPassword1" />
    </Form>
  </Story>
</Canvas>

### Required Label

By setting `isRequired` to `true`, it is possible to provide a visual cue to the user about the requirement.

<Canvas>
  <Story name="Required Label">
    <Form>
      <Form.Label htmlFor="inputPassword2" isRequired>
        Password
      </Form.Label>
      <Form.Control type="password" id="inputPassword2" />
    </Form>
  </Story>
</Canvas>

### Visually Hidden Label

Hides the label visually while still allowing it to be read by assistive technologies.

<Canvas>
  <Story name="Visually Hidden Label">
    <Form>
      <Form.Label htmlFor="inputPassword3" isVisuallyHidden>
        Password
      </Form.Label>
      <Form.Control type="password" id="inputPassword3" />
    </Form>
  </Story>
</Canvas>

### Form.Label Props

**We are working to display the props in a table. Until then, you can find the props below:**

```typescript
export interface FormLabelProps {
  /** Content of button */
  children?: React.ReactNode;
  /** Uses controlId from <FormGroup> if not explicitly specified. */
  htmlFor?: string;
  /** Hides the label visually while still allowing it to be read by assistive technologies */
  isVisuallyHidden?: boolean;
  /** Applies 'require' styling to form label */
  isRequired?: boolean;
  /** Allows for the customization of "required" label. It is helpful for translations. Default: "required"*/
  requiredText;
  /** Additional custom classNames */
  className?: string;
}
```
