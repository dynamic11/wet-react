# Forms

## Form Check

For the non-textual checkbox and radio controls, FormCheck provides a single component for both types that adds some additional styling and improved layout.

### Required

You can set a checkbox or radio as required using the `isRequired` prop.

By default it will show the "required" styling but you can set the visibility using `showRequiredStyling`

You can also customize the required message using the `requiredText` prop

<Canvas>
  <Story name="Required">
    <Form>
      <Form.Check
        type="checkbox"
        label="required field with visible styling"
        isRequired
      />
      <Form.Check
        showRequiredStyling={true}
        isRequired
        type="checkbox"
        label="required field with visible styling"
        requiredText="custom message"
      />
      <Form.Check
        showRequiredStyling={false}
        isRequired
        type="checkbox"
        label="required field with hidden styling"
      />
      <Form.Check
        isRequired
        type="radio"
        label={`radios do not show styling`}
      />
    </Form>
  </Story>
</Canvas>

### Inline

Group checkboxes or radios on the same horizontal row by adding the `isInline` prop.

### Form.Select Props

The `<Form.Check>` component accepts all of the default props for the `check` HTML element in addition to the ones in the table below.

More info about the HTML element:<br/>
https://developer.mozilla.org/en-US/docs/Web/API/HTMLInputElement

**We are working to display the props in a table. Until then, you can find the props below:**

```typescript
export interface FormCheckProps {
  /** label attribute */
  label?: React.ReactNode;
  /** The HTML input type, which is only relevant if as is 'input' (the default). */
  type?: typeType;
  /** Make the control disabled. */
  isDisabled?: boolean;
  /** Groups controls horizontally with other <Form.Checks>. */
  isInline?: boolean;
  /** Add "aria-invalid="true" to input */
  isInvalid?: boolean;
  /** Is field required */
  isRequired?: boolean;
  /** Show the "required styling" for the field */
  showRequiredStyling?: boolean;
  /** Allows for the customization of "required" label. It is helpful for translations. Default: "required" */
  requiredText?: string;
  /** Uses controlId from <FormGroup> if not explicitly specified. */
  id?: string;
  /** Name of form field. */
  name?: string;
  /** Additional custom classNames */
  className?: string;
}
```
