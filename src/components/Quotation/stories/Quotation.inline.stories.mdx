# Quotation

Use inline quotes for snippets of content that are quoted from another source. They appear within a sentence. <br/>
Inline Quotations (") can be added using `&quot;<q>`  and  `&quot;</q>`


## Example

<Canvas>
  <p>In this sentence, he said &quot;<q>Quoted sentence&quot;</q> and I heard it.</p>
</Canvas>
