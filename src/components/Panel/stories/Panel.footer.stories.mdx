# Panel

## Panel with a footer

You can add an optional footer to the `Panel` by using setting the `footerContent` prop.

<Canvas>
  <Story
    name="Panel With Footer"
    args={{
      headerContent: 'Panel header',
      footerContent: 'Panel with footer',
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## Panel without a footer

You can also leave `footerContent` empty to hide the footer.

<Canvas>
  <Story name="Panel Without Footer">{Template.bind({})}</Story>
</Canvas>
