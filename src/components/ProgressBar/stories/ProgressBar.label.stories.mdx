# ProgressBar

## ProgressBar with label

Add a label prop to show a visible percentage. For low percentages, consider adding a min-width to ensure the label's text is fully visible.

<Canvas>
  <Story name="With label" args={{ now: 60, label: '60%' }}>
    {Template.bind({})}
  </Story>
</Canvas>

## ProgressBar with no label

The label is optional and can be omitted

<Canvas>
  <Story name="With no label" args={{ now: 60 }}>
    {Template.bind({})}
  </Story>
</Canvas>
