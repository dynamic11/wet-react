# Table Styling

There are several ways to change a Table's appearance and make it more readable for users.

## Striped

Set the `striped` prop to true to give the table stripes

<Canvas>
  <Story name="Striped" args={{ striped: true }}>
    {Template.bind({})}
  </Story>
</Canvas>

## Hover

Use the `hover` prop so the row the user has their mouse on is highlighted.

<Canvas>
  <Story name="Hover" args={{ hover: true }}>
    {Template.bind({})}
  </Story>
</Canvas>

## Responsive

The `responsive` prop makes wide tables more manageable by enabling horizontal scrolling of the table.

<Canvas>
  <Story name="Responsive" args={{ responsive: true }}>
    {LargeTemplate.bind({})}
  </Story>
</Canvas>

## Bordered

Use the `bordered` prop to better seperate data in different cells.

<Canvas>
  <Story name="Bordered" args={{ bordered: true }}>
    {Template.bind({})}
  </Story>
</Canvas>

## Condensed

Use the `condensed` prop to reduce the height of rows.

<Canvas>
  <Story name="Condensed" args={{ condensed: true }}>
    {Template.bind({})}
  </Story>
</Canvas>
