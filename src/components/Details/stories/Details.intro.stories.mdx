# Details

Build vertically collapsing component.

## Example

Click the summaries below to expand/collapse the detail content.

<Canvas>
  <Details
    data={[
      {
        key: 'key1',
        title: 'Summary 1',
        children: <h2>This is Summary content</h2>,
        className: '',
      },
      {
        key: 'key2',
        title: 'Summary 2',
        children: <h2>This is Summary content 2</h2>,
      },
      {
        key: 'key3',
        title: 'Summary 3',
        children: <h2>This is Summary content as well 3</h2>,
      },
    ]}
  />
</Canvas>

## Available Props

The list is autogenerated based on the array of `detailData` in the `data` prop.

The general structure of `data` is defined as follows:

```js
data = [{
  key: string;
  title: ReactNode;
  children: ReactNode;
  className?: string;
}]
```

The `<Details>` component accepts all of the default props for the `details` HTML element in addition to the ones in the table below.

More info about the HTML element:<br/>
https://developer.mozilla.org/en-US/docs/Web/API/HTMLDetailsElement
