# Tabs

Use to provide contextual feedback messages for typical user actions. Choose from a handful of available and flexible alert messages.

## Example

<Canvas>
  <Tabs
    defaultActiveKey="key2"
    data={[

        className: '',
      },
      {
        key: 'key2',
        title: 'Tab 2',
        children: <h2>This is tab content 2</h2>,
      },
      {
        key: 'key3',
        title: 'Tab 3',
        children: <h2>This is tab content as well 3</h2>,
      },
    ]}
  ></Tabs>
</Canvas>
