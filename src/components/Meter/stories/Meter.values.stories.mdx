# Meter

Using the `max` and `min` props it is possible to set the upper and lower bounds of the `Meter`

## Min Value

The **minimum** value that the meter starts from can be set using the `min` prop.

In this example the `min` is set to `90`, the `max` defaults to `100`, and the value is `95` so the meter is half full.

<Canvas>
  <Story name="Min Value" args={{ min: 90, value: 95 }}>
    {Template.bind({})}
  </Story>
</Canvas>

## Max Value

The **maximum** value that the progress bar reaches can be set using the `max` prop.

In this example the `max` is set to `50`, the `min` defaults to `0`, and the value is `25` so the meter is half full.

<Canvas>
  <Story name="Max Value" args={{ max: 50, value: 25 }}>
    {Template.bind({})}
  </Story>
</Canvas>

## Max & Min values

The **maximum** and **minimum** values can be set at the same time.

In this example the `min` is set to `10`, the `max` is set to `20`, and the value value is `15` so the meter is half full.

<Canvas>
  <Story name="Max/Min Values" args={{ min: 10, max: 20, value: 15 }}>
    {Template.bind({})}
  </Story>
</Canvas>
