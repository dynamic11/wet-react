## Heading Levels

Generate headings to structure a webpage. You can select the heading levels `h1-h6` by using the `level` prop. It will auto generate html tags `<h1>`to`<h6>`.

**The default level is `h1`.**

## Example

<Canvas>
  <Title level="h1">h1. Heading 1</Title>
</Canvas>
