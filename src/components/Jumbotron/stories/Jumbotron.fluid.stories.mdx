export const Template = ({ children, ...rest }) => {

    <Jumbotron {...rest}>

    </Jumbotron>

# Jumbotron

Use a Jumbotron's `fluid` attribute to change its appearance.

## Fluid Jumbotron

Make a Jumbotron fluid so that it takes up all of it's container's space.

<Canvas>
  <Story name="Fluid" args={{ fluid: true }}>
    {Template.bind({})}
  </Story>
</Canvas>

## Non Fluid Jumbotron

A non-fluid Jumbotron will not take up all the space by default.

<Canvas>
  <Story name="Not fluid" args={{ fluid: false }}>
    {Template.bind({})}
  </Story>
</Canvas>
