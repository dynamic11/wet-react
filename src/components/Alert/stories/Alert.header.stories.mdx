# Alert

## Alert with a header

You can add an optional header to the `Alert` by using `<Alert.Header>`

<Canvas>
  <Story name="With Header">{WithHeaderTemplate.bind({})}</Story>
  <Story name="No Header">{NoHeaderTemplate.bind({})}</Story>
</Canvas>

## Custom heading level

You can set the heading level of the `Alert` by using the `level` prop for `<Alert.Header>`. The default value is `h2`.

Possible obtions are: `h1`, `h2`, `h3`, `h4`, `h5`, `h6`
