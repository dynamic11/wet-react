# Links

## Alert with links

You can add links to the `<Alert>` by using `<Alert.Link>`. The styling of the link is dependent on the `variant` chosen for the `<Alert>`.

<Canvas>
  <Story name="No Link">{NoLinkTemplate.bind({})}</Story>
  <Story name="With Link">{WithLinkTemplate.bind({})}</Story>
  <Story name="With Multiple Links">{WithMultipleLinksTemplate.bind({})}</Story>
</Canvas>

## Custom Link Destination (`href`)

The `href` that the link is points to can be defined using the `href` prop for the `<Alert.Link>`.
